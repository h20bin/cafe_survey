# ☕ 목포시민 카페 이용 실태 및 만족도 조사

## 📌 프로젝트 소개

**목포시민 카페 이용 실태 및 만족도 조사**는 Streamlit을 활용하여 제작한 설문조사 웹 애플리케이션입니다.

목포시민의 카페 이용 현황과 만족도를 조사하고, 수집된 데이터를 그래프로 시각화하여 분석할 수 있도록 구현하였습니다.

---

## 🎯 개발 목적

* 목포시민의 카페 이용 패턴 조사
* 카페 이용 목적 및 소비 성향 분석
* 카페 이용 만족도 및 불편사항 수집
* 설문 결과를 통계 자료로 시각화

---

## ✨ 주요 기능

### 사용자

* 설문 참여
* 카페 이용 실태 조사
* 설문 결과 저장(CSV)

### 관리자

* 관리자 로그인(비밀번호: `1234`)
* 총 응답자 수 확인
* 문항별 통계 그래프 확인
* 서술형 응답 조회
* 응답 데이터 CSV 다운로드

## ▶ 실행 방법

### 1. 저장소 복제

```bash
git clone https://github.com/사용자이름/저장소이름.git
cd project
```

### 2. 라이브러리 설치

```bash
python -m pip install -r requirements.txt
```

### 3. 실행

```bash
python -m streamlit run app.py
```
